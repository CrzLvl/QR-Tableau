Queensland Rail Network Map
  • Created in Tableau Public.
  • Geographical data prepared based on TransLink GTFS Real-Time Feed.
  • Ride time data prepared based on TransLink Train Timetables.
This project demonstrates familiarity with Tableau, including preparing, importing, displaying and designing data, sheets and dashboards. Each line is coloured as per the officially designated line colours for improved readability. Line geographical shapes and their associated properties (line name, ride time, etc.) are represented in a GeoJSON file sizing in at 702 KB, resulting in quick loading and rendering.
